# choosen-for-weekends
# 🎡 다국어 스타일 휠 | 多语言风格罗盘

> 오늘의 스타일을 결정하는 재미있는 휠 | 趣味转盘决定今日风格

---

## ✨ 기능 | 功能

- **🎯 오늘의 스타일 | 今日风格**  
  휠을 돌려 오늘의 스타일 결정 | 转动转盘随机决定

- **🎲 취미 랜덤 생성 | 随机生成爱好**  
  6~8개의 새로운 취미 + 스타일 조합 | 6-8个兴趣 × 随机风格

- **📅 무제한 기록 | 无限记录**  
  모든 회전 자동 저장 | 每次转动自动保存

- **🌐 3개국어 지원 | 三语支持**  
  中文 / English / 한국어

---

## 🚀 시작하기 | 开始使用

`index.html` 실행 | 直接打开即可

### 조작법 | 操作说明

| 한국어 | 中文 |
|--------|------|
| 휠 클릭 또는 버튼 누르기 | 点击转盘或按钮 |
| [취미 랜덤 생성] 클릭 | 点击[随机生成爱好] |
| 언어 버튼 클릭 | 点击语言按钮 |

---

## 📁 파일 | 文件

- `index.html` - 전체 앱 | 完整应用

---

## 🔧 커스터마이징 | 自定义

코드 내 배열 수정 | 修改以下数组：

- `INTEREST_BASE` - 취미 목록 | 兴趣爱好库
- `STYLE_SUFFIX` - 스타일 접미사 | 风格后缀库
